蒼の彼方のフォーリズム -世界への答え- 日本語化パッチ

対応バージョン
  aokana_asuka_doujin-1.0-pc

導入方法
  1. 本パッチを展開します。
  2. JP_Patch.exe を起動します。
  3. 元ゲームのフォルダを選択します。
     選択するフォルダには「game」フォルダと
     「aokana_asuka_doujin.exe」がある必要があります。
  4. インストールします。
     追加CGを指定画像に差し替える場合は、カスタムインストールを選び、
     「追加CGを指定画像に差し替える」を有効にしてください。
  5. 元ゲームの aokana_asuka_doujin.exe を起動すると、日本語で始まります。

手動導入
  JP_Patch\game フォルダ内の内容を、元ゲームフォルダの
  game フォルダへ同じ構成のまま上書きコピーしてください。

注意
  - 本パッチにはゲーム本体、音声、シナリオ原文は含まれていません。
    タイトル画像および設定画面の日本語化に必要なUI差分画像のみを含みます。
  - 元ゲームの正規配布元・制作者のガイドラインに従って利用してください。
  - 既存のセーブデータは念のためバックアップを推奨します。
  - 追加CGの差し替えは任意です。通常インストールでは元のCGを変更しません。

内容
  - シナリオ全文の日本語ローカライズ
  - 基本UIの日本語化
  - 日本語タイトル画面
  - 任意の追加CG差し替え（抱擁CG2枚、桜の明日香CG1枚）

ソースからのビルド
  Inno Setup 6 を導入後、配布フォルダ直下で次を実行します。

  & "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" "JP_Patch.iss"

  JP_Patch_Output\JP_Patch.exe が生成されます。
