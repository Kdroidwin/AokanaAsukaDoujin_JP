# 苍之彼方的四重奏：交予世界的答卷（蒼の彼方のフォーリズム -世界への答え-）日本語化パッチ

![Screenshot](https://raw.githubusercontent.com/Kdroidwin/AokanaAsukaDoujin_JP/main/JP_Patch/game/tl/japanese/4a1cc2ff-e402-4904-a683-aecce21decbe.png)

### 対応バージョン
- `aokana_asuka_doujin-1.0-pc`

---

# 導入方法

1. 本パッチを展開します。
2. `JP_Patch.exe` を起動します。
3. 元ゲームのフォルダを選択します。  
   選択するフォルダには、以下が存在する必要があります。
   - `game` フォルダ
   - `aokana_asuka_doujin.exe`
4. インストールします。
   - 追加CGを指定画像に差し替える場合は、**カスタムインストール**を選択し、
     **「追加CGを指定画像に差し替える」** を有効にしてください。
5. 元ゲームの `aokana_asuka_doujin.exe` を起動すると、日本語でプレイできます。

---

# 手動導入

`JP_Patch/game` フォルダ内の内容を、元ゲームフォルダ内の `game` フォルダへ、同じフォルダ構成のまま上書きコピーしてください。

---

# 注意

- 本パッチには以下は含まれていません。
  - ゲーム本体
  - 音声
  - シナリオ原文

  日本語化に必要なタイトル画像および設定画面のUI差分画像のみを収録しています。

- 元ゲームの正規配布元・制作者のガイドラインに従って利用してください。

- 既存のセーブデータは、念のためバックアップを推奨します。

- 追加CGの差し替えは任意です。通常インストールでは元のCGは変更されません。

# あらすじ
「空を見ろ。空を見続けろ。答えはそこにある。」

かつて先生が晶也に残した言葉。

けれど、その言葉だけでは足りなかった。
その一言だけでは、晶也を深い闇の底から救い上げることはできなかった。

しかし、夕暮れの空の下、一人の少女が、優しく、それでいて少しだけわがままに、晶也自身も気づかなかった本当の願いを口にする。

「昌也くんが空を飛ぶ姿を見てみたいです。」

「見せてください。晶也くんがどんな選手なのか、私は知りたいんです。」

晶也はもう、背中にまとわりつく黒い泥を恐れない。
夕焼けの空の下、空を翔ける天才としてのすべてを彼女へ見せた。

「空を見ろ。空を見続けろ。空を楽しめ。答えはそこにある。」

それは晶也が彼女へ贈った言葉。
自分のように空に押し潰されることなく、空そのものを楽しんでほしいという願いだった。

――でも、それだけでよかったのだろうか。

世界の頂点に立ったとき、何を手に入れたのか。

そして、何を失ったのか。

その問いを突きつけられたとき、明日香はどう答えるのだろう。

「空を見ろ。空を見続けろ。空を楽しめ。――○○、空を。答えはそこにある。」

これは、私たちが世界へ届ける、最後の答え。

これで――きっと、よかったんだよね。

# ソースからのビルド

`Inno Setup 6` をインストール後、配布フォルダ直下で以下を実行してください。

```powershell
& "C:\Program Files (x86)\Inno Setup 6\ISCC.exe" "JP_Patch.iss"
```

ビルド完了後、以下の場所にインストーラーが生成されます。

```text
JP_Patch_Output/JP_Patch.exe
```
